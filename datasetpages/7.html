Arenas

Custodian

Engineering - Geomatics

Data Currency Comments

These point files contain the Name, Address, URL and X & Y Coordinates of each of the City's owned and operated arenas.

Dataset Description

The coordinates are for the approximate location of the arena.

Data Accuracy Comments

The attributes include the Name, Address, URL and Northing & Easting or Latitude/Longitude depending on file format.

Attributes

SHP formatted data are projected in UTM, NAD83, Zone 17 Metric and LL84. CSV formatted data is projected in LL84. DWG file is projected in UTM, NAD83, Zone 17 Metric. KMZ files have automatic transformation when applied.
